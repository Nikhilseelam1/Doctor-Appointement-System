# Doctor Appointment System

## Overview

Doctor Appointment System is a web-based application designed to streamline the process of booking and managing medical appointments. The system allows patients to schedule appointments with doctors, while administrators can manage doctors, patients, and appointment records efficiently.

This project demonstrates full-stack development concepts including authentication, database integration, CRUD operations, and role-based access.

---

## Problem Statement

In many small clinics and hospitals, appointment scheduling is still handled manually. This leads to:

- Double bookings
- Long waiting times
- Poor record management
- Difficulty in tracking patient history

This system digitizes the process and improves overall efficiency.

---

## Key Features

Patient Features:
- Register and login securely
- View available doctors
- Book appointments
- View appointment history

Doctor Features:
- View assigned appointments
- Manage availability
- Update appointment status

Admin Features:
- Add, update, and remove doctors
- Manage patient records
- Monitor all appointments
- Maintain system data

## System Workflow

1. User registers or logs into the system.
2. Patient selects a doctor and chooses an available time slot.
3. Appointment is stored in the database.
4. Doctor and admin can view and manage the appointment.
5. Status updates are reflected in real time.
